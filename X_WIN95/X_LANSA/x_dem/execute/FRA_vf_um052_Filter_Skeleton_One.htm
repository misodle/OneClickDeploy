Quoi ? Cet assistant génère le squelette du code pour un critère de recherche. Le code produit nécessite des modifications manuelles. Il a été conçu pour être utilisé par des développeurs ayant créé des filtres auparavant et ne souhaitent qu'un squelette de filtre pour y ajouter leur code eux-mêmes.   
Qui ? Développeurs expérimentés 
Où ? Plateformes Navigateur Web (génère du code RDMLX)
Comment ?
Cliquer le bouton Suivant >> en bas de ce formulaire. 

Puis précisez le protocole d'identification de votre objet métier et le fichier physique sous-jacent.

Précisez également le module de Service que vous allez utiliser pour accéder au fichier. 

Utilisez l'assistant d'aide (F1) pour plus d'information à propos des protocoles d'identification des objets métier.

Cliquer le bouton Suivant >>.

 
Choisir parmi les différentes options ce qui définit le type de squelette à produire.   

Cliquer le bouton Suivant >>.  

 

 
Cliquer le bouton Générer le code pour générer le code du critère de recherche.

 
Entrez un nom et une description et cliquer sur le bouton Créer pour créer directement un nouveau réutilisable à partir du code généré,

ou

Cliquer the Copier le code vers le presse-papier puis collez-le dans un réutilisable (avec plateforme cible navigateur Web) que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
