What? This assistant produces the code for a filter that allows the user to search for information using any of the logical views of a specified physical file. 

Filters are used to get data and load it into instance lists (e.g. lists of Customers, lists of Products, lists of Orders, lists of Employees, etc).

Typically Visual LANSA Framework filters are presented like this example (in the area circled in red):

 
Who? New Developers 
Where? Windows platforms (generates RDMLX)
How?
Click the Next >> button at the bottom of this form.

Choose the fields containing the data that will be loaded onto the instance list. 


Click the Next >> button.
Choose all the logical views that the user will be able to search on.

Click the Next >> button.
Check any other options you want.

Click the Generate Code button to generate the filter code.
Enter a name and description and click the Create button to directly create a new VL component from the generated code,
or,
Click the Copy Code to Clipboard button and then paste the source into a VL component that you have already created.
Note! If you need more details about any option while you are using this assistant click on the option and then use F1 to display the associated help text details.
