Quoi ? Cet assistant génère le squelette du code pour un gestionnaire d'action. Le code produit nécessite des modifications manuelles. Il a été conçu pour être utilisé par des développeurs ayant créé des gestionnaires d'action auparavant et ne souhaitent qu'un squelette de gestionnaire d'action pour y ajouter leur propre code eux-mêmes.  
Qui ? Développeurs expérimentés 
Où ? Plateforme Navigateur (génère du code RDML WAM)
Comment ?
Cliquez le bouton Suivant >> en bas de ce formulaire.  

Puis choisissez parmi les options proposées le type de squelette de gestionnaire d'action devant être généré.   

Utilisez l'assistant d'aide (F1) pour plus d'informations sur les options proposées.

 
Cliquez le bouton Générer le code pour générer le code du gestionnaire d'action.

 
Entrez un nom et une description et Créer un nouveau WAM à partir du code généré, ou Copier le code vers le presse-papier puis collez-le dans un WAM que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
