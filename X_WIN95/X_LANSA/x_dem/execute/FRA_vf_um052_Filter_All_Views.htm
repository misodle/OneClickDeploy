Quoi ? Cet assistant génère le code pour un critère de recherche permettant à l'utilisateur de rechercher des informations dans n'importe quelle vue logique d'un fichier physique particulier. 

Les filtres sont utilisés pour lire des données et les charger dans des listes d'instances (exple liste de clients, listes de produits, listes de commandes, listes de salariés, etc).

Typiquement les critères de sélection du Visual LANSA Framework sont présentés comme dans cet exemple (à l'intérieur du cercle rouge):

 
Qui ? Nouveaux Développeurs 
Où ? Plateforme Windows (génère du code RDMLX)
Comment ?
Cliquez le bouton Suivant >> en bas de ce formulaire.

Choisissez les zones contenant les données à charger dans la liste d'instances. 


Cliquez le bouton Suivant >>.
Choisissez toutes les vues logiques que l'utilisateur pourra utiliser pour ses recherches.

Cliquez le bouton Suivant >>.
Choisissez toute autre option nécessaire.

Cliquez le bouton Générer le code pour générer le code du critère de recherche.
Entrez un nom et une description et Créer un nouveau composant VL à partir du code généré, ou Copier le code vers le presse-papier puis collez-le dans un composant VL que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
