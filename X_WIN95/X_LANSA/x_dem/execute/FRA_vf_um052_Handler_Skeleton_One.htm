Quoi ? Cet assistant génère le squelette du code pour un gestionnaire d'action.  Le code produit nécessite des modifications manuelles.    

Les gestionnaires d'action des objets métier exécutent une commande sur une ou plusieurs instances d'objet métier (exemple Afficher un Employé, montrer l'Historique d'un Produit, Imprimer toutes les Commandes sélectionnées).

Dans beaucoup de Framework Visual LANSA les gestionnaires de commande sont présentés à l'utilisateur comme ceci (dans la zone entourée de rouge) :

Toutefois il y a beaucoup d'autres façons de présenter les gestionnaires d'action. Utilisez l'assistant d'aide pour plus d'information et trouver des exemples de gestionnaires d'action.

Puis précisez le protocole d'identification de votre objet métier et le fichier physique sous-jacent.

Utilisez l'assistant d'aide (F1) pour plus d'information à propos des protocoles d'identification des objets métier.

Cliquer le bouton Suivant >>.

 
Choisir parmi les différentes options qui définissent les types de fonctionnalités que votre gestionnaire de commande doit supporter.   

Utilisez l'assistant d'aide (F1) pour plus d'information à propos des options affichées sur ce formulaire.

 
Cliquer le bouton Générer le code pour générer le code du gestionnaire de commande.

 
Entrez un nom et une description et cliquer sur le bouton Créer pour créer directement un nouveau réutilisable à partir du code généré,

ou

Cliquer le bouton Copier le code vers le presse-papier puis collez-le dans un réutilisable (avec plateforme cible navigateur Web) que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
