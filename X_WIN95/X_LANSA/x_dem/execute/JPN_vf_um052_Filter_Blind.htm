機能 このCRUDウィザードは、隠れたビジネス・オブジェクト用のコードを生成します。
対象者 新規の開発者
環境 Windowsプラットフォーム (RDMLXコードを生成)
詳細 CRUDウィザードは、フィルタ及びコマンド・ハンドラの迅速な生成を手助けするように設計されています。コマンド・ハンドラは、共通コードやシステム定義テーブルに格納された情報のCRUD(Create/Read/Update/Delete)で必要になるものです。これらのタイプのテーブルは、多くの商用アプリケーション(例：会社、 通貨、郵便番号、地域等)に存在し、一般には列が少なく、更新の度合いが低いです。CRUDウィザードは、これらのタイプのテーブルの保守を行うアプリケーションの構築過程をスピードアップさせます。CRUDウィザードを使用した推奨開発サイクルは、以下のようになります。
  • コード テーブルと呼ばれるアプリケーションを定義します。
  • 各共通コードまたはシステム定義テーブルの場合：

    => ビジネス・オブジェクト(例：通貨、郵便番号)を定義します。

    => フィルタをひとつだけ定義します。

    => コマンドを、新規追加、コピー、明細及び削除のみ有効にします。

    => Windows及びBrowserプラットフォームで繰り返します(必要に応じて)：

    • このウィザードを使用してビジネス・オブジェクト向けのフィルタ・プログラムを作成し、スナップインします。
    • CRUD コマンドハンドラ ウィザードを使用してシングル・コマンド・ハンドラ・プログラムを作成します。シングル・コマンド・ハンドラを、新規作成、コピー、明細及び削除の各コマンドにスナップインします(例：4つのコマンドすべてに、ハンドラとして同じプログラムを使用します)。
使用方法
このフォームの下部にある次へ >>ボタンをクリックします。

その後、ビジネス・オブジェクトの識別プロトコルとビジネス・オブジェクトを支える物理ファイルを指定します。

ビジネス・オブジェクトの識別プロトコルの詳細については、ヘルプアシスタント(F1キー押下)を利用して下さい。

 

コードの生成ボタンをクリックして、フィルタ・コードを生成します。

 

クリップボードへコードをコピーボタンを使用します。

 

生成したコードをVisual LANSAエディタ内のフィルタ・プログラムにペーストします。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1 キーを押してヘルプアシスタントを表示して下さい。
