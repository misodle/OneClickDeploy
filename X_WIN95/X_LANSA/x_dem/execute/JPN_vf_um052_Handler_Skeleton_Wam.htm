機能 このウィザードは、コマンド・ハンドラ向けのスケルトン・コードを生成します。生成されたコードから、自分自身で内容を完成させる必要があります。これは、以前にコマンド・ハンドラを作成し、スケルトン・コマンド・ハンドラを使用して自身で特化したコードを手動で追加したいだけの開発者が使用するために設計されたものです。
対象者 経験豊富な開発者
環境 ブラウザプラットフォーム(WAM RDMLコードを生成)
使用方法
このフォームの下部にある次へ >>ボタンをクリックします。

その後、生成するスケルトン・コマンド・ハンドラ・コードのタイプを定義づける各種オプションから選択します。

表示されるオプションの詳細については、いずれもヘルプアシスタント (F1キー押下)を利用して下さい。

 

コード生成ボタンをクリックして、アウトライン・フィルタ・コードを生成します。

 

クリップボードへコードをコピーボタンを使用します。

 
生成したコードをVisual LANSAエディタ内のコマンド・ハンドラ・プログラムにペーストします。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1 キーを押してヘルプアシスタントを表示して下さい。
