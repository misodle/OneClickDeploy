What? This assistant creates a code fragment that uses the instance list manager method #avListManager.AddtoList to add to or update an entry in the business object instance list. 
Who? Experienced developers
Where? Windows platforms (generates RDMLX code)
How?
Click the Next >> button at the bottom of this form. 

Then specify your business object's identification protocol and the physical file that underpins the business object. 

Use F1 for more information about business object identification protocols. 

 
Click the Generate Code button to generate the filter code.

 
Use the Copy Code to Clipboard button.

 
Paste the generated code at the appropriate point into the object you are editing in the Visual LANSA editor.
Note! If you need more details about any option while you are using this assistant click on the option and then use F1 to display the associated help text details.
