What?

Cet assistant génère le code d'un gestionnaire d'action permettant à un utilisateur final d'envoyer des données vers MS-Excel via un fichier CSV.

L'utilisateur final peut choisir les zones qu'il veut envoyer parmi une liste de zones.
Quand l'utilisateur clique sur "Envoyer à MS-Excel" les données envoyées à MS-Excel sont basées soit sur ce qui est dans la liste d'instances, ou sur ce qui est sélectionné dans la liste d'instances.

Le gestionnaire d'action généré ressemble à l'exemple affiché ici (encadré de rouge)

La feuille de calculs générée ressemblera à ceci :

 
Qui ? Nouveaux développeurs 
Où ? Plateformes Windows (génère du code RDMLX)
Comment ?
Cliquez le bouton Suivant >> en bas de ce formulaire. 

Puis précisez le protocole d'identification de votre objet métier. 

Utilisez l'assistant d'aide (F1) pour plus d'information à propos des protocoles d'identification des objets métier.

Cliquez le bouton Suivant >>.

 
Sélectionnez les zones parmi lesquelles l'utilisateur final pourra choisir. Définissez les dans la zone nommée:

"Zones devant apparaître en haut du gestionnaire d'action"
Cliquez le bouton Générer le code pour générer le code du gestionnaire d'action.

 
Entrez un nom et une description et Créer un nouveau composant VL à partir du code généré, ou Copier le code vers le presse-papier puis collez-le dans un composant VL que vous avez déjà créé.
Note! Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
