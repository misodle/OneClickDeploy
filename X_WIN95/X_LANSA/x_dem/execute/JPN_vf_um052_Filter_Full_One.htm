機能 このウィザードは、指定の物理ファイルまたは論理ビューを用いて情報を検索するフィルターのコードを生成します。

フィルターを利用して、ビジネス オブジェクトのインスタンスリスト(例: 顧客リスト、 製品リスト、注文リスト、社員リスト 等) を動的に生成できます。

標準では Visual LANSA フレームワークのフィルターは、以下の例のように表示されます (赤で囲んだ部分):

 
対象者 新規の開発者
環境 ブラウザ プラットフォーム (Visual LANSA のブラウザ コード (RDMLX) を生成)
使用方法
このフォームの下にある [次へ ] をクリックします。

その後、ビジネス オブジェクト識別プロトコルとこのビジネス オブジェクトを支える物理ファイルを指定します。

ファイルのアクセスに使用するサービス モジュールも指定します。

ビジネス オブジェクト識別プロトコルの詳細については、ヘルプ アシスタント (F1) を利用して下さい。

[次へ] をクリックします。

 
フィルターで検索する物理ファイルか論理ビューを選択し、検索オプションと利用する制限を設定します。

 

 

 
[コード生成] をクリックし、フィルターのコードを生成します。

 
名前と記述を入力し、[生成] をクリックして生成されたコードから新規再利用可能パーツを直接作成する、

もしくは

[クリップボードへコピー] をクリックし、既に作成されている再利用可能パーツにソースコードを貼り付けます (ターゲット プラットフォームは Web ブラウザ) 。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1キーを押してヘルプアシスタントを表示します。
