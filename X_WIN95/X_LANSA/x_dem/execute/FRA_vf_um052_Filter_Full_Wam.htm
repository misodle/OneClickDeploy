Quoi ? Cet assistant génère le code pour un critère de recherche dans un fichier physique ou une vue. 

Les filtres sont utilisés pour créer dynamiquement les listes d'instances d'un objet métier (exple liste de clients, de produits, de commandes, de salariés, etc..).

Typiquement les critères de sélection du Visual LANSA Framework sont présentés comme dans cet exemple (à l'intérieur du cercle rouge) :

 
Qui ? Nouveaux développeurs 
Où ? Plateforme Navigateur (génère du code RDML WAM)
Comment ?
Cliquez le bouton Suivant >> en bas de ce formulaire.

Puis précisez le protocole d'identification de votre objet métier. 

Utilisez l'assistant d'aide (F1) pour plus d'information à propos des protocoles d'identification des objets métier.

Cliquez le bouton Suivant >> en bas de ce formulaire.  

 
Choisir le fichier physique ou la vue logique lus par le filtre et choisir les options de recherche et les limites à utiliser.     

 

 

 
Cliquez le bouton Générer le code pour générer le code du critère de recherche.

 
Entrez un nom et une description et Créer un nouveau WAM à partir du code généré, ou Copier le code vers le presse-papier puis collez-le dans un WAM que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
