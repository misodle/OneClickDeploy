Quoi ? Cet assistant crée un fragment de code. Le code généré est une routine de méthode uExecute du gestionnaire d'action (MthRoutine) qui gère des instances multiples d'un objet métier. 

Par exemple, si vous voulez créer un gestionnaire d'action pour imprimer le détail de, par exemple, tous les produits actuellement sélectionnés, vous pouvez utiliser cet assistant. Il créera le code de la MthRoutine incluant une boucle dans laquelle les infos des produits sélectionnés seront extraites.  

 
Qui ? Développeurs expérimentés
Où ? Plateformes Windows (génère du code RDMLX)
Comment ?
Cliquez sur le bouton Suivant >> en bas de ce formulaire. 

Puis précisez le protocole d'identification de votre objet métier et le fichier physique sous-jacent à l'objet métier. 

Utilisez l'assistant d'aide (F1) pour plus d'information à propos des protocoles d'identification des objets métier. 

 
Cliquez le bouton Générer le code pour générer le code du gestionnaire d'action.

 
Entrez un nom et une description et Créer un nouveau composant VL à partir du code généré, ou Copier le code vers le presse-papier puis collez-le dans un composant VL que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
