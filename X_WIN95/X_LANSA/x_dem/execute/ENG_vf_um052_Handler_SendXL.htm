What?

This assistant produces the code for a command handler that allows the end user to send data to MS-Excel as a CSV file.

The end user can choose the fields they want to send from a list of fields.
When the end user presses "Send to MS-Excel" the data sent to MS-Excel is based on either what is in the instance list, or what is selected in the instance list.

How?
Click the Next >> button at the bottom of this form. 

Then specify your business object's identification protocol. 

Use F1 for more information about business object identification protocols. 

Click the Next >> button.

 
Choose the fields that the end user will be able to select from. Specify them in the area called:

"Fields that you want to appear on the top of your command handler"
Click the Generate Code button to generate the command handler code.

 
Enter a name and description and click the Create button to directly create a new VL component from the generated code,
or,
Click the Copy Code to Clipboard button and then paste the source into a VL component that you have already created.
