Quoi ? Cet assistant produit un composant utilisé dans un gestionnaire de commande produit par l'assistant de code Gestionnaire de commande qui maintient une liste Partie 1.
Qui ? Nouveaux Développeurs 
Où ? Plateformes Windows (génère RDMLX)
Comment ?
Cliquer le bouton Suivant >> en bas de ce formulaire.
Choisir les zones que vous voulez voir dans la liste du gestionnaire de commande, et quelles zones vous voulez pouvoir modifier.

Cliquer le bouton Générer le code pour générer le code du filtre.
Entrez un nom et une description et cliquer sur le bouton Créer pour créer directement un nouveau réutilisable à partir du code généré,

ou,

Cliquer sur le bouton Copier le code vers le presse-papier puis collez-le dans un réutilisable (avec plateforme cible navigateur Web) que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
