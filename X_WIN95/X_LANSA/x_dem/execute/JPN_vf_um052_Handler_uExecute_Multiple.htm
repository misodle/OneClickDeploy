機能 このウィザードは、 コード フラグメントを作成します。生成されたコードはuExecuteコマンド・ハンドラ・メソッド・ルーチン(例：MthRoutine)であり、これは、ビジネス・オブジェクトの複数インスタンスを処理するものです。

例えば、コマンド・ハンドラを作成して、現在選択しているすべての製品の詳細を印刷したかった場合に、このウィザードが使用できるかもしれません。このウィザードではMthRoutineコードとその内部にループを作成して、現在選択しているすべての製品の詳細を抽出します。

 
対象者 経験豊富な開発者
環境 Windowsプラットフォーム(RDMLXコードを生成)
使用方法
このフォームの下部にある次へ >>ボタンをクリックします。

その後、ビジネス・オブジェクトの識別プロトコルとビジネス・オブジェクトを支える物理ファイルを指定します。

ビジネス・オブジェクトの識別プロトコルの詳細については、ヘルプアシスタント (F1キー押下)を利用して下さい。

 
コード生成ボタンをクリックして、フィルタ・コードを生成します。

 
クリップボードへコードをコピーボタンを使用します。

 
適切なポイントに生成したコードを、Visual LANSAエディタで編集するコマンド・ハンドラにペーストします。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1 キーを押してヘルプアシスタントを表示して下さい。
