Quoi ? Cet assistant génère le code pour un gestionnaire de base d'objet métier. En fonction des options que vous utiliserez vous aurez ou pas à ajouter du code manuellement ensuite.    

Les gestionnaires d'action d'objet métier exécutent une action sur une ou plusieurs instances d'objet métier (exple. Afficher un Salarié, afficher l'Historique d'un Produit, Imprimer toutes les Commandes sélectionnées).

Toutefois il y a d'autres façons de présenter les gestionnaires d'action. Reportez-vous à l'assistant d'aide pour plus d'informations et d'exemples à propos des gestionnaires d'action.
