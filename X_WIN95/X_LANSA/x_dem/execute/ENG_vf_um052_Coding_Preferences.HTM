What? This assistant sets your LANSA code generation preferences
Who? For all developers
Where? For all platforms
How?
Click the Next >> button at the bottom of this form.

The resulting form will display and allow you to alter your current code generation preferences.

 
Then click the Generate Code button to see a generated code sample that uses your preferences.  
Note! If you need more details about any option while you are using this assistant click on the option and then use F1 to display the associated help text details.
