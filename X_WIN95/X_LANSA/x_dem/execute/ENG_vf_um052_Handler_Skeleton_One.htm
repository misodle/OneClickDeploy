What? This assistant produces the code skeleton for a command handler.  You will need to complete it manually.    

Business object command handlers execute a command against one or more selected business object instances (e.g. Display an Employee, show the History of a Product, Print all selected Orders).

In a many Visual LANSA framework command handlers are presented to the user like this (in the area circled in red):

However there are many other ways that command handlers may be presented. Use F1 for more details and examples of command handlers.

 
Who? New Developers 
Where? Windows platforms (generates RDMLX code)
How?
Click the Next >> button at the bottom of this form. 

Then specify your business object's identification protocol and the physical file that underpins the business object.

Also specify the Server Module you are going to use to access the file. 

Use F1 for more information about business object identification protocols. 

Click the Next >> button.

 
Choose from various options that define what type of features your command handler should support.    

Use F1 for more information about the options shown on this form.

 
Click the Generate Code button to generate the command handler code.

 
Enter a name and description and click the Create button to directly create a new reusable part from the generated code,

or,

Click the Copy Code to Clipboard button and then paste the source into a reusable part (with Target platform Web browser) that you have already created.
