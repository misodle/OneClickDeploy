機能 このアシスタントは、ビジネス・オブジェクトに従属するデータを操作するためのコマンドハンドラーを提供します。

コマンドハンドラーとは次のようなものです。(赤で囲った部分):

コマンドハンドラーは子データのリストを含み、ユーザーがそのリストからエントリーを選択した時、そこでデータを保守できます。

この例では、親ビジネスオブジェクトは社員（PSLMSTに基づく）で、子データは社員スキル（PSLSKLに基づく）です。コマンドハンドラーは選択された社員の全スキルを表示します。
対象者 新規の開発者 
環境 Windowsプラットフォーム (RDMLXを生成)
使用方法
フォームの下部にある 次へ >> ボタンをクリックします。

親ビジネスオブジェクトの AKey と NKey を選択します。
(コマンドハンドラーがインスタンスリストから受け取る情報)


次へ >> ボタンをクリックします。

子データを読むのに必要なファイル、ビュー、キーを選択します。

次へ >> ボタンをクリックします。

子オブジェクトのリスト内で表示したいフィールドと編集したいフィールドを選択します。

コード生成 ボタンをクリックし、 フィルターのコードを生成します。
名前と記述を入力し、 生成 ボタンをクリックし、生成されたコードから直接 VL コンポーネントを作成する、
または、
クリップボードへコピー ボタンをクリックし、ソースを 先に作成しておいた VL コンポーネントに貼り付けます。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1キーを押してヘルプアシスタントを表示して下さい。
