Welcome to the Online Reporting Demonstration

Note: This example uses the Microsoft XML parser. You might receive a fatal error upon selecting a Group. Please, refer to the documentation provided with the VL Framework for details on how to download a suitable version of the MSXML parser.

1. Select the Group you want to report from.

2. If required, input your Report Content Selection Criteria.

3. Review the resulting report in this window.

4. Change the style and content of the report dynamically.

Note: See shipped demonstration component DF_DET18 for technical details.
