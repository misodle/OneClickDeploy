What? This CRUD assistant produces the code for a hidden business object filter.   
Who? New Developers 
Where? Windows platforms (generates RDMLX)
Details? The CRUD assistants were designed to help you rapidly produce the filters and command handlers required for the CRUD (Create/Read/Update/Delete) of information stored in common code or system definition tables. These types of tables are found in many commercial applications (e.g. Companies, Currencies, Zip Codes, Location Codes, etc) and typically have few rows and a low update frequency. The CRUD assistants will speed up the process of building maintenance applications for these types of tables. The recommended development cycle using CRUD assistants goes something like this:
  • Define an application called Code Tables (say)
  • For each common code or system definition table:

    => Define a business object (e.g. Currencies, Zip Codes)

    => Define just one filter.

    => Enable the commands New, Copy, Details and Delete only.

    => Repeat for Windows and Browser platforms (as appropriate):

    • Create a filter program for the business object using this assistant. Snap it in.
    • Create a single command handler program using the CRUD command handler assistant. Snap the single command handler into each of the New, Copy, Details and Delete commands (i.e. the same program is used as the handler for all four commands).   
How?
Click the Next >> button at the bottom of this form.

Choose the file containing the data.

Then drag and drop the fields you want to load on the instance list.
(Drag the fields you want to show on the list, and also the unique identifier fields. )

(If you want the instance list to be changed to match your selection, check the checkbox.) 


Click the Next >> button.
Check any other options you want.

Click the Generate Code button to generate the filter code.
Enter a name and description and click the Create button to directly create, compile, and snap in a new VL component from the generated code,
or,
Click the Copy Code to Clipboard button and then paste the source into a VL component that you have already created.
Note! If you need more details about any option while you are using this assistant click on the option and then use F1 to display the associated help text details.
