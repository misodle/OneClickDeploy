What? This assistant produces a command handler that maintains child data belonging to the business object.

The command handler looks like this (in the area circled in red):

The command handler contains its own list of child data, and when the user selects an entry from this list, they can maintain the data on it.

In the example shown, the parent business object is Employees (based on PSLMST), and the Child data is Employee skills (based on PSLSKL). The command handler shows all the skills for the selected employee.
Who? New Developers 
Where? Windows platforms (generates RDMLX)
How?
Click the Next >> button at the bottom of this form.

Choose the Akey and NKey fields for the parent business object.
(The information the command handler receives from the instance list)

Click the Next >> button.
Choose which fields you want in the command handler list, and which fields you want to edit.
