Quoi ? Cet assistant génère le code pour un critère de recherche dans un fichier physique ou une vue.

Les filtres sont utilisés pour créer dynamiquement les listes d'instances d'un objet métier (exemple : liste de clients, de produits, de commandes, de salariés, etc...).

Typiquement les critères de sélection du Visual LANSA Framework sont présentés comme dans cet exemple (à l'intérieur du cercle rouge) :

 
Qui ? Nouveaux Développeurs 
Où ? Plateforme navigateur Web (génère du code navigateur Visual LANSA (RDMLX))
Comment ?

Cliquer le bouton Suivant >> en bas de ce formulaire. 


Choisir le fichier contenant les données et le module serveur utilisé pour y accéder.

Puis faire glisser les zones que vous voulez charger dans la liste d'instance. 

(Faire glisser les zones à afficher sur la liste, et également les zones identifiant unique.) 

(Si vous voulez que la liste soit modifiée pour refléter votre sélection, cochez la case)

 
Cliquer le bouton Suivant >>.
Choisir le fichier physique ou la vue logique lus par le filtre et choisir les options de recherche et les limites à utiliser.     

Cliquer le bouton Suivant >>.


Cochez toute autre option souhaitée.
Cliquer le bouton Générer le code pour générer le code du critère de recherche. 
Entrez un nom et une description et cliquer sur le bouton Créer pour créer directement un nouveau réutilisable à partir du code généré,

ou

Cliquer Copier le code vers le presse-papier puis collez-le dans un réutilisable (avec plateforme cible navigateur Web) que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
