What? This assistant produces the skeleton code for a filter. The code produced requires significant manual completion. It is designed for use by developers who have created filters before and only want a skeleton filter to manually add their own specialized code to.   
Who? Experienced Developers 
Where? Web browser platforms (generates RDMLX code)
How?
Click the Next >> button at the bottom of this form. 

Then specify your business object's identification protocol and the physical file that underpins the business object.

Also specify the Server Module you are going to use to access the file. 

Use F1 for more information about business object identification protocols. 

Click the Next >> button.

 
Choose from various options that define what type of skeleton code should be produced.   

Click the Next >> button.  

 

 
Click the Generate Code button to generate the outline filter code.

 
Enter a name and description and click the Create button to directly create a new reusable part from the generated code,

or,

Click the Copy Code to Clipboard button and then paste the source into a reusable part (with Target platform Web browser) that you have already created.
Note! If you need more details about any option while you are using this assistant click on the option and then use F1 to display the associated help text details.
