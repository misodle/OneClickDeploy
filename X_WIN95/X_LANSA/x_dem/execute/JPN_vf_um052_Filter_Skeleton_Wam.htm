機能 このウィザードは、フィルタ向けのスケルトン・コードを生成します。生成されたコードから、自分自身で内容を完成させる必要があります。これは、以前にフィルタを作成し、スケルトン・フィルタを使用して自身で特化したコードを手動で追加したい開発者が使用するために設計されたものです。
対象者 経験豊富な開発者
環境 ブラウザプラットフォーム(WAM RDMLコードを生成)
使用方法
コード生成ボタンをクリックして、アウトライン・フィルタ・コードを生成します。

 
クリップボードへコードをコピーボタンを使用します。

 
生成したコードをVisual LANSAエディタ内のフィルタ・プログラムにペーストします。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1 キーを押してヘルプアシスタントを表示して下さい。
