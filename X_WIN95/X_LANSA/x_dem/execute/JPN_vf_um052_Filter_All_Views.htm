機能 このアシスタントは、 ユーザーが指定された物理ファイルの論理ビュー使用して、情報を検索できるようフィルタのコードを生成します。 

フィルタは、データを取得し、それを インスタンスリスト にロードします (例　顧客のリスト、 製品のリスト、注文のリスト、社員のリスト etc)。

標準では Visual LANSA フレームワーク フィルタは、以下の例のように表示されます (赤で囲った部分):

 
対象者 新規の開発者 
環境 Windows プラットフォーム (RDMLX を生成)
使用方法
フォームの下部にある 次へ >> ボタンをクリックします。

インスタンスリストにロードされるデータを含むフィールドを選択します。 


次へ >> ボタンをクリックします。
ユーザーがそれに基づいて検索できる 全ての論理ファイルを選択します。

次へ >> ボタンをクリックします。
必要なオプションを選択します。

コード生成 ボタンをクリックし、 フィルタのコードを生成します。
名前と記述を入力し、生成 ボタンをクリックし、生成されたコードから直接新しい VL コンポーネントを作成する、
または、
クリップボードへコピー ボタンをクリックし、ソースを 先に作成しておいた VL コンポーネントに貼り付けます。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1 キーを押してヘルプアシスタントを表示して下さい。
