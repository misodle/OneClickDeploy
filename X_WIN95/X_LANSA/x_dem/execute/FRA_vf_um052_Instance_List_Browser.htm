Quoi ? Cet assistant génère le code à intégrer pour un affichage de liste d'instance.  

Le cadre d'application Visual LANSA propose un affichage de liste d'instance par défaut pour tous les objets métier. Dans de nombreux framework Visual LANSA, les listes d'instances sont présentés à l'utilisateur ainsi (voir à l'intérieur du cercle rouge) : 

  

Vous pouvez remplacer l'affichage standard par le votre. Reportez-vous à l'assistant d'aide pour plus d'informations et d'exemples concernant l'affichage des listes d'instances .     

 
Qui ? Développeurs expérimentés 
Où ? Plateformes Windows (génère du code RDMLX)

Cliquez le bouton Suivant >> en bas de ce formulaire. 

Puis précisez le protocole d'identification de votre objet métier. 

Utilisez l'assistant d'aide (F1) pour plus d'information à propos des protocoles d'identification des objets métier.

 
Cliquez le bouton Générer le code pour générer le code gestionnaire de liste d'instance.

 
Entrez un nom et une description et Créer un nouveau composant VL à partir du code généré, ou Copier le code vers le presse-papier puis collez-le dans un composant VL que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
