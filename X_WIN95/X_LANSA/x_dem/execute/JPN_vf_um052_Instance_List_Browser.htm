機能 このウィザードはスナップイン・インスタンス・リスト・ブラウザのソースを作成します。

Visual LANSA Frameworkは、すべてのビジネス・オブジェクトに対し、デフォルトのインスタンス・リスト・ブラウザを提供します。 標準では、インスタンス・リスト・ブラウザはVisual LANSA Frameworkにおいて、以下のように表示されます(赤で囲んだ部分)。

デフォルトのブラウザは、自分で作成したものに置き換えることができます。 置き換えの方法及びインスタンス・リスト・ブラウザをフレームワークにスナップインするための全詳細は、ヘルプ アシスタントを参照して下さい。

 
対象者 経験豊富な開発者
環境 Windowsプラットフォーム(RDMLXコードを生成)
使用方法
このフォームの下部にある次へ >>ボタンをクリックします。

その後、ビジネス・オブジェクトの識別プロトコルを指定します。

ビジネス・オブジェクトの識別プロトコルの詳細については、ヘルプアシスタント (F1キー押下)を利用して下さい。

 
コード生成ボタンをクリックして、スナップイン・インスタンス・リスト・ブラウザ・コードを生成します。

 
クリップボードへコードをコピーボタンを使用します。

 
生成したコードを、Visual LANSAエディタ内のスナップイン・インスタンス・リスト・ブラウザ・プログラムにペーストします。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1 キーを押してヘルプアシスタントを表示して下さい。
