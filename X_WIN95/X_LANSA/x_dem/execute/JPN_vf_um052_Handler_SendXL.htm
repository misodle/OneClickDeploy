機能

このアシスタントはCSVファイルとしてMS-Excelへデータを送ることをユーザーに許可するためのコードを作成します。

ユーザーはリストから送りたいフィールドを選択することができます。
ユーザーが "MS-Excelへ送信" を押したとき、MS-Excelへ送られるデータはインスタンスリストの中のものか、インスタンスリストで選択されたものに基づきます。

作成されるコマンド・ハンドラーは以下のように表示されます(赤で囲まれた部分)。

作成されるスプレッドシートは以下のようになります：

使用方法
フォームの下部にある 次へ >>ボタンをクリックします。 

その後、ビジネス・オブジェクトの 識別プロトコルを指定します。 

ビジネス・オブジェクトの識別プロトコルの詳細については、 ヘルプアシスタント(F1キー押下)を利用して下さい。 

次へ >> ボタンをクリックします。

 
ユーザーが選択可能なフィールドを選択してください。

"コマンド・ハンドラーの上部に表示したいフィールド"のエリアでそれらのフィールドを指定します。
コード生成 ボタンをクリックして、コマンド・ハンドラー・コードを生成します。

 
名前と記述を入力し、 生成 ボタンをクリックし、生成されたコードから直接 VL コンポーネントを生成する、
または、
クリップボードへコピー ボタンをクリックし、ソースを先に作成しておりた VL コンポーネントに貼り付けます。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1 キーを押してヘルプアシスタントを表示して下さい。
