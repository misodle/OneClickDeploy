What? This assistant produces the code for a snap in instance list browser.  

The Visual LANSA Framework provides a default instance list browser for all business objects. In a typical Visual LANSA Framework the instance list browser appears like this (indicated by the red circled area): 

  

You may replace the default browser with one of your own. For full details of how to do this and how to snap your instance list browser into your framework use F1 for help.
