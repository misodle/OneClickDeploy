Quoi ? Cet assistant CLMS produit le code pour un filtre d'objet métier caché.   
Qui ? Nouveaux développeurs 
Où ? Plateformes Windows (génère du RDMLX)
Détails ? Les assistants CLMS ont été conçus pour vous aider à développer rapidement les filtres et les gestionnaires d'action nécessaires à la Création/Lecture/Modification/Suppression des informations stockées dans le code commun ou les tables de définition du système. Ces types de tables existent dans de nombreuses applications (Sociétés, Devises, Code postaux, etc...) et ont peu de lignes et une faible fréquence de mise à jour. Les assistants CLMS accélèrent le processus de création des applications de maintenance pour ces types de tables. Le cycle de développement recommandé avec ces assistants ressemble à ceci :
  • Définir une application Tables de codes (par exemple)
  • Pour chaque code commun ou table de définition système :

    => Définir un objet métier (exple. Devises, codes postaux)

    => Définir juste un filtre.

    => Activer seulement les actions Nouveau, Copier, Détails et Supprimer.

    => Répéter pour les plateformes Windows et Navigateur (suivant les besoins):

    • Créer un programme critère de recherche pour l'objet métier avec cet assistant. L'intégrer au cadre d'application.
    • Créer un seul programme gestionnaire d'action avec l'assistant de gestionnaire d'action CLMS. Intégrer le gestionnaire d'action pour chaque action Nouveau, Copier, Détails et Supprimer (i.e. le même programme est utilisé comme gestionnaire pour les quatre actions).   
Comment ?
Cliquez le bouton Suivant >> en bas de ce formulaire.  

Choisissez les zones contenant les données à charger dans la liste d'instances. 


Cliquez le bouton Suivant >>.
Choisissez toute autre option nécessaire.

Cliquez le bouton Générer le code pour générer le code du critère de recherche.
Entrez un nom et une description et Créer un nouveau composant VL à partir du code généré, ou Copier le code vers le presse-papier puis collez-le dans un composant VL que vous avez déjà créé.
NB : Si vous souhaitez plus d'informations sur les options possibles quand vous utilisez cet assistant cliquez sur l'option et utilisez F1 pour afficher les informations de l'assistant d'aide.
