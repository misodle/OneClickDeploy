機能 このウィザードでは、ベーシックのビジネス・オブジェクト・コマンド・ハンドラ向けのコードを生成します。利用するオプションによっては、後から手動でコードを追加する必要があります。

ビジネス・オブジェクト・コマンド・ハンドラは、選択されたひとつ以上のビジネスオブジェクトインスタンス(例：社員を表示する、製品の履歴を表示する、選択されたすべてのオーダーを印刷する)に対して、コマンドを実行します。

多くのVisual LANSAフレームワークでは、コマンド・ハンドラは、以下のように表示されます(赤で囲んだ部分)。

しかし、コマンド・ハンドラを表示する方法は、他にもたくさんあります。詳細及びコマンド・ハンドラの例については、ヘルプアシスタントを参照して下さい。

 
対象者 新規の開発者
環境 ブラウザプラットフォーム(WAM RDMLコードを生成)
使用方法
このフォームの下部にある次へ >>ボタンをクリックします。

その後、ビジネス・オブジェクトの識別プロトコルを指定します。

ビジネス・オブジェクトの識別プロトコルの詳細については、ヘルプアシスタント (F1キー押下)を利用して下さい。

次へ >>ボタンをクリックします。

 

コマンド・ハンドラのパネルに表示するフィールドを指定します。

次へ >>ボタンをクリックします。

 

 

 

コマンド・ハンドラがサポートする機能のタイプを定義づける各種オプションから選択します。

このフォームに表示されるオプションの詳細については、ヘルプアシスタント (F1キー押下)を利用して下さい。

 

コード生成ボタンをクリックして、コマンド・ハンドラ・コードを生成します。

 

クリップボードへコードをコピーボタンを使用します。

 
生成したコードをVisual LANSAエディタ内のコマンド・ハンドラ・プログラムにペーストします。
注意 このアシスタントのオプションの詳細を見たい場合は、オプションをクリックし F1 キーを押してヘルプアシスタントを表示して下さい。
